# Guía para Video de Presentación (5–7 minutos)

## Objetivo
Presentar el proyecto SGTURNOS, explicar el problema, objetivos, aplicación de las fases del ciclo de vida, resultados esperados y lecciones aprendidas.

## Estructura sugerida
1. Apertura (10–15s)
   - Nombre del proyecto y equipo.
2. Problema e importancia (30–45s)
   - Dolor actual: mallas manuales, novedades sin trazabilidad.
3. Objetivos y alcance (45–60s)
   - Qué mejora el módulo; áreas impactadas.
4. Fases del ciclo de vida (2–3 min)
   - Inicio, Planificación, Ejecución, Monitoreo y Control, Cierre.
5. Resultados esperados y KPIs (60–90s)
   - Tiempo de aprobación, cobertura, incidencias, satisfacción.
6. Lecciones aprendidas y cierre (30–45s)
   - Validar roles y seguridad temprano; involucrar usuarios.
7. Enlace del video
   - [Por definir: YouTube/Drive]

## Tips de grabación
- Todos los integrantes deben aparecer en cámara.
- Audio claro, entorno silencioso, buena iluminación.
- Diapositivas simples con puntos clave.
- Ensayar con cronómetro para ajustar tiempos.

## Checklist
- [ ] Guion repartido por personas.
- [ ] Prueba técnica (cámara/audio).
- [ ] Slides listas (si aplica).
- [ ] Enlace generado y verificado.
- [ ] Compartir enlace junto con el PDF.
